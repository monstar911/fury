# **Events**

The Bet module emits the following events

## *MsgPlaceBet*

|  Type         |  Attribute Key|    Attribute Value    |
|:-------------:|:-------------:|:---------------------:|
| place_bet     | bet_creator   |  {creator}            |
| place_bet     | bet_uid       |  {bet_uid}            |
| message       | module        |  bet                  |
| message       | action        |  place_bet            |
| message       | sender        |  {creator}            |
